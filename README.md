# Mountain Master

**Dein intelligenter Begleiter für alpine Touren**

Mountain Master ist eine moderne Android-App für ambitionierte Bergsportler, entwickelt mit Jetpack Compose und Kotlin. Die App bietet eine durchdachte Oberfläche für die Tourenplanung und Navigation im alpinen Gelände.

## Technologie-Stack

- **Sprache:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Architektur:** MVVM mit ViewModel
- **Navigation:** Navigation Compose
- **Icons:** Material Icons (Extended)
- **Minimum SDK:** API 24 (Android 7.0)
- **Target SDK:** API 34 (Android 14)
